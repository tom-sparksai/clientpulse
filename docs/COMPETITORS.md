# ClientPulse Competitor Analysis

## Direct Competitors

### 1. Bonsai (hellobonsai.com)
**Target:** Service businesses, freelancers, agencies (10,000+ users)
**Pricing:** ~$25-75/mo
**Features:**
- Client management
- Project management
- Finance management (invoicing, billing)
- Automations
- Reports & KPIs
- Team collaboration

**Strengths:**
- Unified platform approach
- Strong client experience focus
- Comprehensive onboarding

**Weaknesses:**
- Not real-time focused
- Generic approach (not agency-specific)

---

### 2. HoneyBook (honeybook.com)
**Target:** Small businesses, creatives
**Pricing:** ~$19-79/mo
**Features:**
- CRM & lead capture
- Proposals, contracts, invoices in one flow
- Online payments
- Automations
- AI features (email drafts, meeting notes, business trends)
- Project pipeline

**Strengths:**
- AI-powered features
- $12B+ payments processed
- 90% invoices paid on time
- Strong for creatives

**Weaknesses:**
- Focused on creatives, not dev agencies
- No real-time client updates

---

### 3. Basecamp
**Target:** All businesses, project management
**Pricing:** ~$15/user/mo or flat $299/mo
**Features:**
- Project management
- Team chat
- File sharing
- To-dos

**Weaknesses:**
- Not agency/client focused
- No invoicing
- Old UI

---

### 4. Monday.com
**Target:** All businesses, work management
**Pricing:** ~$10-20/seat/mo
**Features:**
- Project boards
- Automations
- Integrations
- Dashboards

**Weaknesses:**
- Too generic
- Complex for simple client work
- No client portal focus

---

### 5. Dubsado
**Target:** Service professionals, agencies
**Pricing:** ~$20-40/mo
**Features:**
- Client management
- Contracts & forms
- Invoicing
- Client portals
- Workflows

**Strengths:**
- Client portal focus
- Agency-oriented

---

## ClientPulse Differentiation

### Our Unique Value:
1. **Real-time Everything** - Live project updates via Supabase Realtime
2. **Zero-friction Client Portal** - Token-based access, no login required
3. **Built for Dev Agencies** - Understands software project workflows
4. **Modern UI/UX** - Linear/Notion quality, not enterprise bloat
5. **Simple Pricing** - Flat rate, no per-seat pricing

### Target Positioning:
> "The real-time client dashboard built specifically for software agencies"

### Pricing Strategy (Suggested):
- **Starter:** $49/mo - 5 clients, 1 user
- **Growth:** $99/mo - 25 clients, 5 users
- **Agency:** $199/mo - Unlimited clients, unlimited users

### Key Differentiators to Emphasize:
1. Clients see updates LIVE (no email updates needed)
2. In-context chat per project
3. Clean, modern UI (not enterprise ugly)
4. Built by a dev agency (Sparks AI) for dev agencies
5. Self-hostable with Supabase
